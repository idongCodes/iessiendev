My Journey

Searching for an alternative way to learn and commit to making programming my career, I found bootcamps. Taking advantage of a fresh start, I enrolled into Lambda School. Since January 12, 2020 - I’ve created landing pages with HTML, designed with CSS and it’s technologies, like grid and flex-box. I’ve built single page apps using React, designed with LESS, styled-components and SCSS. This knowledge was expanded on by creating SPAs connected to back-ends with Node and express(.js), allowing for user authentication.

Also, I learned to manage state using React’s useState and useEffect hooks as well as Redux for handling global application state. I overcame a major struggle learning to implement binary tree breadth first and depth first traversal algorithms. I’m grateful to have a bit of experience in the agile process - working on cross-functional teams of over five engineers from varying backgrounds, building products for real world clients, breaking down feature releases into user stories and then further into engineering tasks, with each user story being a deliverable per one week Sprint.

My exposure to technologies such as Trello, Slack and Zoom, is also important as these aid in team work flow. Lest I forget, I’ve also gained experience using Git commands for cloning repositories, fetching branches, pulling from branches, creating new branches, switching between branches, committing and pushing to branches and resolving merge conflicts. Moreover, I’ve been able to take advantage of peer programming as well as peer code reviews. Humbly speaking, I’ve come quite a way from where I started and I’m proud of the man I am today as a result of my past. I am excited to continue down this path of constant growth and learning.
